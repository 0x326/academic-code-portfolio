Web Filter
==========

Requires Python 3.6

Running
-------

::

   pip3 install pipenv
   pipenv install
   pipenv run python http_filter_proxy.py

Documentation
-------------

Written with Sphinx_.
View the documentation at ``docs/_build/html/index.html``
or build it yourself with::

   pipenv install --dev
   cd docs/
   pipenv run make html
   # Open _build\html
